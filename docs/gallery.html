ğŸ“ How to Add Your Artwork

Ready to add your own artwork to the gallery? It's easy! Here's how:

  1. Save your image: Put your image file in the src/images/ folder
  2. Add to gallery: Open gallery.html and add this code in the gallery-grid section: 
<div class="gallery-item">
  <img src="images/your-image.jpg" alt="Description of your artwork" />
</div>

Pro tip: Always include a good description in the alt attribute. It helps describe your artwork to everyone!
